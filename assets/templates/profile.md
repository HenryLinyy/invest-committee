# 投資人設定檔（invest-committee 讀取）

| 欄位 | 值 | 說明 |
|---|---|---|
| 總投資資金 | 未設定 | 例：NT$ 2,000,000（設定後部位建議才會給出金額） |
| 單一部位上限 | 10% | 單一個股佔總資金的最大比例 |
| 風險承受度 | 中 | 低／中／高（影響部位調整係數，見 stage-action.md） |
| 預設投資週期 | 波段（1-6個月） | 短線（<1個月）／波段（1-6個月）／長線（>6個月） |
| 基準指數 | 台股加權指數 | 報告中相對強弱的比較基準 |
| 資料收集模型 | haiku | 抓資料的子 agent 用哪個省錢模型：haiku（最省）／sonnet（較會判斷網搜來源）。分析仍用你主 session 的模型（建議 Opus）。留空＝haiku |

> 修改本檔後，下次分析自動生效。「總投資資金」留「未設定」時，部位建議只給 %，不給金額。
> 省 token：階段 1 抓資料委派給上面的省錢模型，主 session 用強模型只做分析——所以請把你的 Claude Code session 設成 Opus/Sonnet 跑分析，讓 skill 把便宜的抓取活外包出去。
