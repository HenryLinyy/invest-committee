# 階段 2：基本面與成長品質（口徑）

方法論源自「科技股分析」skill，通用化為多市場適用。所有數值必附來源；取不到標「未獲取」。

## 成長韌性

**5年營收 CAGR** ＝ `((第5年營收 / 第1年營收)^(1/4) − 1) × 100%`
**5年淨利 CAGR** ＝ 同式，用歸母淨利（虧損年導致無法開根時標「不適用（區間內有虧損）」並改用逐年描述）
**正成長年數** ＝ 5 年內 YoY 為正的年數（0-5），營收與淨利各算一個。

判定表：

| CAGR | 正成長年數 | 判定 |
|---|---|---|
| >15% | ≥4 | 持續穩定成長 |
| >15% | ≤3 | 週期性爆發 |
| 5-15% | ≥4 | 穩健成長 |
| <5% 或負 | 近 2 年轉正 | 階段性恢復 |
| <5% 且無改善 | — | 成長停滯 |

資料不足 5 年（新上市）→ 用可得年數並標註「上市未滿5年，以 N 年計」。

## 獲利品質

| 指標 | 口徑 | 判讀 |
|---|---|---|
| 毛利率趨勢 | 近 8 季毛利率方向 | 連 3 季上升＝改善；連 3 季下滑＝警戒（提示產品競爭力或成本壓力） |
| 淨利現金含量 | 經營活動現金流淨額 ÷ 歸母淨利（年度） | ≥1 健康；0.5-1 觀察；<0.5 警戒（獲利含金量低，注意應收/存貨堆積） |
| 存貨周轉率 | 營業成本 ÷ 平均存貨 | 與同業中位數對照，明顯偏低＝去化風險 |

## 研發強度（科技股適用）

研發收入比 ＝ 研發費用 ÷ 營業收入。>5% 具成長底蘊；>15% 高強度（同時提醒商業化回款壓力）。非科技股跳過本項並註明。

## 估值位階

1. 現值 PE / PB 落在**自身近 5 年分位數**：<20% 低位、20-80% 中位、>80% 高位。
2. 與**同業中位數**對照（同業取 3-5 家代表性公司）。
3. 高成長股 PE 高位時補充 PEG（PE ÷ 預估盈餘成長率）視角：<1 相對合理。

## 本階段固定小結格式（供階段 6 引用）

```
成長韌性：〔判定〕（營收CAGR x.x%／正成長 n 年；淨利CAGR x.x%／正成長 n 年）
獲利品質：〔健康/觀察/警戒〕（毛利率趨勢＋現金含量數值）
估值位階：〔低位/中位/高位〕（PE x.x 落在 5 年 p 分位；同業中位數 y.y）
```
