# 階段 4：資金籌碼與技術擇時（口徑）

方法論融合「量化股票分析」與「股票個股分析 v3」。

## 執行順序

1. 依 data-sources.md §5 組 K線 JSON（建議 ≥120 交易日；長線 250 日）。
2. 執行 `python3 <skill路徑>/scripts/indicators.py --data_file <json路徑>`。
3. 依本檔口徑解讀腳本輸出。**指標數值一律取腳本輸出，禁止心算。**

## 指標解讀門檻表

| 指標 | 腳本輸出鍵 | 解讀 |
|---|---|---|
| RSI(14) | `technical_indicators.rsi` | ≥80 嚴重超買／≥70 超買／≤30 超賣／≤20 嚴重超賣 |
| KDJ | `technical_indicators.kdj` | J>100 超買、J<0 超賣；金叉/死叉訊號 |
| MACD | `technical_indicators.macd` | 金叉/死叉、紅綠柱放大縮小（動能方向） |
| BOLL | `technical_indicators.boll` | 帶寬>20 波動劇烈、<5 醞釀變盤；%B 位置 |
| 乖離率 | `technical_indicators.bias`（bias_5…bias_250） | 5/10日 0~3% 強勢未過熱；>8% 短線過熱；60/120日為負＝中長期超跌 |
| 量比 | `volume_analysis.volume_ratio` | 1.2-2 溫和放量健康；>3 過熱警戒；<0.5 量能萎縮 |
| 換手率 | 成交量 ÷ 流通股數（需另取流通股數） | 1-5% 溫和；>10% 籌碼鬆動 |
| ATR% | `technical_indicators.atr.atr_pct` | 波動基準，供 stage-action.md 停損計算 |
| 均線排列 | `trend_analysis.daily` | 多頭/空頭排列、短期向上/向下、震盪 |
| 多週期共振 | `trend_analysis.resonance` | 共振一致時標註「多週期共振，可信度較高」 |
| 支撐壓力 | `support_resistance` | 前高前低＋斐波那契，含強度標註 |
| 缺口 | `gap_analysis` | 未回補向上缺口＝支撐；向下缺口＝壓力；含類型（突破/持續/竭盡/普通） |

## 資金面口徑（依市場）

| 市場 | 口徑 |
|---|---|
| 台股 | 三大法人買賣超**連續性**（連 3 日同向為有效訊號；外資＋投信同向最強）、外資持股比變化、融資使用率變化（融資大增＋股價滯漲＝散戶接刀警訊） |
| 美股 | 機構持股比變化（富途，季頻）、成交量結構（放量上漲/下跌） |
| 港股/A股 | 富途資金流向（OpenD 開啟時）；退回成交量結構分析 |

## 「資金-趨勢-效率」三位一體診斷

| 組合 | 判讀 |
|---|---|
| 資金連續流入 ＋ 低乖離（bias_20 < 3%） | 低位吸籌／蓄勢，安全邊際較高 |
| 資金連續流入 ＋ 量比溫和（1.2-2）＋ 價漲 | 控盤推升，趨勢延續機率高 |
| 資金流入 ＋ 高換手（>10%）＋ 高振幅 | 跟風過熱，防短期回撤 |
| 資金流出 ＋ 高乖離（bias_20 > 8%） | 高位出貨警訊 |
| 資金流出 ＋ 低乖離 ＋ 縮量 | 觀望盤整，等趨勢訊號 |

禁止孤立解讀單一指標——結論必須是組合判定。

## 本階段固定小結格式（供階段 6 引用）

```
趨勢狀態：〔判定〕（日線排列＋週線＋共振結論）
資金狀態：〔三位一體判定〕（法人/資金連續性數據）
關鍵價位：支撐 x.x（強度）／壓力 y.y（強度）／ATR% z.z
```
