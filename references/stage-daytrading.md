# 當沖模式口徑（stage-daytrading.md）

當沖＝日內或隔日沖的短線交易視角。**限台股**（twstock 有當沖名單資料）。日K-based：本模式用日K＋法人日報推導「當沖視角」，**不含分時、五檔、內外盤、券商分點**；要 tick 級請接富途 OpenD。紅線黑名單與 §4 資料紀律全部適用。

## §1 當沖適性 Gate（前置硬檢查，🔴 顯性）

分析開始先查三項（ToolSearch 載入）：
- `get_daily_day_trading_targets`（當日可現股當沖名單：該股在不在？）
- `get_suspended_day_trading_announcement`（暫停現股當沖預告）
- `get_market_disposal_stocks`／`get_otc_disposal_stocks`（處置股；處置期間常限制當沖：預收款、人工撮合）

| 結果 | 行為 |
|---|---|
| ✅ 在可當沖名單、無暫停、非處置 | 正常出完整當沖卡＋多空劇本 |
| ⚠️ 暫停當沖預告／處置股／不在可現沖名單 | **🔴 報告頂部警示「本標的目前不可（或受限）當沖」**；只給技術面觀察，**不生成進出場劇本** |
| 查無名單資料 | 標「當沖適性未獲取，下單前請向券商確認」；劇本照出但附此警語 |

## §2 決戰帶（當沖核心概念）

當日多空分界的關鍵價區，勝負在此。取法（來源＝indicators.py 的 `support_resistance` 與 `gap_analysis`）：
- 最近**強壓力**或**強支撐**（strength=強）為決戰帶中樞。
- 若附近有**未回補缺口**，以缺口邊緣為決戰帶邊界（缺口常是當沖攻防點）。
- 決戰帶以「a–b 價區」表示；站上／跌破帶量確認才視為方向成立（做劇本不做信仰）。

## §3 日內停損（嚴於波段）

```
日內停損 ≤ 0.5 × ATR（絕對值，取 indicators.py 的 atr）
```

輸出換算成元與 %。明顯緊於波段的 2×ATR——當沖錯了要快跑。

## §4 當沖適性門檻（可做／觀望）

| 指標 | 可做 | 觀望 |
|---|---|---|
| 當日成交值 | 流動性充足（相對該股常態放大或持平） | 過低＝「量縮非好獵場，等放量」 |
| 量比（`volume_analysis.volume_ratio`） | 1.0–2 溫和 | <0.5 縮量觀望；>3 過熱防追高 |
| ATR%／5日均振幅（`amplitude_5_pct`） | 有波動空間才有價差可做 | 振幅過小＝無肉；漲跌停鎖死＝無法進出 |

三者綜合定性；量縮或鎖漲跌停 → 明說當日非好獵場。

## §5 顏色慣例（僅當沖輸出）

台股 **紅漲綠跌**：dashboard CSS `--up:#c0392b（紅）`、`--down:#1a7f37（綠）`。與波段版（漲跌用綠/紅相反）刻意區隔——當沖看盤者習慣台股紅漲綠跌。

## §6 能力邊界聲明（固定輸出，不可省）

> 本報告基於日K＋法人日報，不含分時、五檔、內外盤、券商分點等 tick 級當沖資訊；如需即時分時資料請接富途 OpenD。技術指標由 scripts/indicators.py 計算。

## §7 風險邊界（當沖＝高風險，強化）

- 不喊單、不代下當沖單（紅線黑名單）。
- **「做劇本、不做信仰」**：劇本的觸發條件不成立就不做，不硬凹方向。
- 固定警語：「市場有風險，當沖尤甚。」
- **單日虧損上限建議 ≤ 總資金 1%**（讀 profile 總資金換算金額）。
- 部位集中度：留倉市值超過 profile 單一部位上限 → 頂部 banner 警示（含超標倍數與 ATR% 換算的單日資金晃動）。

## §8 固定小結格式（供劇本與 dashboard 引用）

```
波動流動性：〔可做/觀望〕（ATR% x.x／5日均振幅 y.y／量比 z.z／成交值）
技術決戰帶：a–b（站上/跌破帶量確認方向）；日內停損 ≤0.5×ATR＝c 元
當沖劇本：多方〔觸發→目標→停損〕｜空方〔觸發→目標→停損〕｜當日適性一句話
```
